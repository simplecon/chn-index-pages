# chn-index-pages

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容涉及通用技术展示、信息索引及前端资源整理，不针对任何特定域名、网站或商业用途。

## 项目简介

本仓库旨在提供一个集中存储与分发 HTML 页面的空间。所有页面均为独立文件，可直接通过 GitHub Pages 或其他静态托管服务访问。适合用于技术演示、学习参考或临时信息发布。

## 目录结构

```
chn-index-pages/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹或文件对应一个独立页面，命名遵循内容主题。
- `assets/` 目录用于存放页面间共享的样式、脚本或多媒体资源。

## 页面归档说明

- 所有页面均为纯 HTML 实现，不依赖后端服务。
- 页面内容可能包含示例数据、静态演示或技术文档片段，不保证长期更新。
- 归档目的为保留历史版本或提供临时访问入口，使用者可自行拷贝或修改。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件即可查看。
3. 若需部署到 GitHub Pages，请确保仓库设置中已启用 Pages 功能，并指向 `main` 分支的根目录或 `docs/` 文件夹。
